### generator weex project cli

``` bash
# 生成weex项目工程
 mcp create <name>

# 下载依赖
 cd <name>
 mcp install

# 启动项目
 mcp start

```
### 打包zip

``` bash
 # 打包之前修改lib/config.json
 mcp build
 
````

### 版本

``` bash
# 1.0.6
 npm run dev
 npm run serve

# 1.2.0

# 1.2.5
```
